# My dwm dotfiles for fedora linux
### **Installation**:
1. Add [terra repoes](https://terra.fyralabs.com/)
2. Install dependencies: ```sudo dnf install -y brightnessctl pavucontrol flameshot feh alacritty fish picom fastfetch jetbrains-mono-fonts starship slock eza Thunar @development-tools gcc make xorg-x11-xinit xorg-x11-xauth libX11-devel libXft-devel libXinerama-devel xorg-x11-server-Xorg xsetroot fontawesome-6-free-fonts```
3. Clone repo: ```git clone https://github.com/Commmdi/dwm_dotfiles```
4. Move everything from dwm_dotfiles directory to home directory: ```mv * ~/ ```
5. Compile dwm in ~/.config/dwm/ directory
6. Reboot
